== Description ==

This plugin will display your Amazon wishlist, you can configure amazon domain, list order, Layout,  and much more. for bugs or features contact: altmannmarcelo@gmail.com

== Installation ==

1. Upload `Amazon-wishlist-pro.php` and `simple_html_dom.php` to the `/wp-content/plugins/Amazon-wishlist-PRO` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Amazon Wishlist PRO and set up your credentials
4. Activate the Amazon Wishlist Pro Widget

== Frequently asked questions ==

= Can I have different Layouts? =

Yes, you can, just need to input the html code on layout area.

= Plugin is displaying removed items, why? =

You probably have the cache option enabled, this option cache your wish list for 1 day to avoid request amazon every time you page receive a hit. Try to invalidate cache.

= How can I invalidate cache?  =

There is an option on pluging settings named `Invalidate cache`, set it to on, and hit update, it will remove the cache file, forcing it to be recreated on next hit.
